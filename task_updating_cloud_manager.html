    жӯЈеңЁжӣҙж–°Cloud Manager

    05/03/2022

    жӮЁеҸҜд»Ҙе°ҮCloud Managerжӣҙж–°зӮәжңҖж–°зүҲжң¬гҖҒжҲ–дҪҝз”ЁNetAppдәәе“ЎиҲҮжӮЁе…ұдә«зҡ„дҝ®иЈңзЁӢејҸгҖӮ

    е•ҹз”ЁиҮӘеӢ•жӣҙж–°

    Cloud ManagerеҸҜеңЁж–°зүҲжң¬жҺЁеҮәжҷӮиҮӘеӢ•иҮӘиЎҢжӣҙж–°гҖӮеҰӮжӯӨеҸҜзўәдҝқжӮЁеҹ·иЎҢзҡ„жҳҜжңҖж–°зүҲжң¬гҖӮ

    еҰӮжһңжІ’жңүеҹ·иЎҢд»»дҪ•дҪңжҘӯгҖҒCloud ManagerжңғеңЁеҚҲеӨң12пјҡ00иҮӘеӢ•жӣҙж–°гҖӮ

    жӯҘй©ҹ

    1. еңЁ Cloud Manager дё»жҺ§еҸ°еҸідёҠи§’гҖҒжҢүдёҖдёӢгҖҢиЁӯе®ҡгҖҚең–зӨәгҖҒз„¶еҫҢйҒёеҸ–гҖҢ * Cloud Manager иЁӯе®ҡ * гҖҚгҖӮ

    2. йҒёеҸ–гҖҢAutomatic Cloud Manager UpdatesпјҲиҮӘеӢ•Cloud Managerжӣҙж–°пјүгҖҚдёӢж–№зҡ„ж ёеҸ–ж–№еЎҠгҖҒз„¶еҫҢжҢүдёҖдёӢгҖҢ* SaveпјҲ*е„Іеӯҳпјү

    е°ҮCloud Managerжӣҙж–°зӮәжңҖж–°зүҲжң¬

    жӮЁжҮүи©Іе•ҹз”ЁCloud Managerзҡ„иҮӘеӢ•жӣҙж–°гҖҒдҪҶжӮЁеҸҜд»ҘйҡЁжҷӮзӣҙжҺҘеҫһз¶Іи·Ҝдё»жҺ§еҸ°жүӢеӢ•жӣҙж–°гҖӮCloud ManagerжңғеҫһAWSдёӯNetAppж“Ғжңүзҡ„S3е„ІеӯҳеҚҖеҸ–еҫ—и»ҹй«”жӣҙж–°гҖӮ

    жӮЁжҮүи©Іе·Із¶“жӘўй–ұйҒҺ "зүҲжң¬зҡ„ж–°еҠҹиғҪ" жүҫеҮәж–°зҡ„йңҖжұӮе’Ңж”ҜжҸҙи®ҠжӣҙгҖӮ

    и»ҹй«”жӣҙж–°йңҖиҰҒе№ҫеҲҶйҗҳзҡ„жҷӮй–“гҖӮжӣҙж–°жңҹй–“з„Ўжі•дҪҝз”ЁCloud ManagerгҖӮ

    жӯҘй©ҹ

    1. жҹҘзңӢдё»жҺ§еҸ°еҸідёӢи§’гҖҒжӘўжҹҘжҳҜеҗҰжңүеҸҜз”Ёзҡ„ж–°зүҲжң¬пјҡ

    2. еҰӮжһңжңүеҸҜз”Ёзҡ„ж–°зүҲжң¬гҖҒи«ӢжҢүдёҖдёӢ* Timeline *д»ҘеҲӨж–·жҳҜеҗҰжңүд»»дҪ•е·ҘдҪңеңЁйҖІиЎҢдёӯгҖӮ

      еҰӮжһңжңүд»»дҪ•е·ҘдҪңжӯЈеңЁйҖІиЎҢдёӯгҖҒи«Ӣзӯүеҫ…е·ҘдҪңе®ҢжҲҗгҖҒз„¶еҫҢеҶҚз№јзәҢдёӢдёҖжӯҘгҖӮ

    3. еңЁдё»жҺ§еҸ°еҸідёӢи§’гҖҒжҢүдёҖдёӢ*гҖҢNew version AvailableгҖҚпјҲеҸҜз”Ёзҡ„ж–°зүҲжң¬пјү*гҖӮ

    4. еңЁCloud Managerи»ҹй«”жӣҙж–°й ҒйқўдёҠгҖҒжҢүдёҖдёӢжӮЁиҰҒзҡ„зүҲжң¬ж—ҒйӮҠзҡ„*жӣҙж–°*гҖӮ

    5. е®ҢжҲҗзўәиӘҚе°Қи©ұж–№еЎҠгҖҒз„¶еҫҢжҢүдёҖдёӢ*зўәе®ҡ*гҖӮ

    Cloud Managerжңғй–Ӣе§Ӣжӣҙж–°зЁӢеәҸгҖӮжӮЁеҸҜд»ҘеңЁе№ҫеҲҶйҗҳеҫҢзҷ»е…Ҙдё»жҺ§еҸ°гҖӮ

    дҪҝз”Ёдҝ®иЈңзЁӢејҸжӣҙж–°Cloud Manager

    еҰӮжһңNetAppиҲҮжӮЁе…ұдә«дҝ®иЈңзЁӢејҸгҖҒжӮЁеҸҜд»ҘзӣҙжҺҘеҫһCloud Managerз¶Іи·Ҝдё»жҺ§еҸ°гҖҒдҪҝз”ЁйҡЁйҷ„зҡ„дҝ®иЈңзЁӢејҸдҫҶжӣҙж–°Cloud ManagerгҖӮ

    дҝ®иЈңзЁӢејҸжӣҙж–°йҖҡеёёйңҖиҰҒе№ҫеҲҶйҗҳзҡ„жҷӮй–“гҖӮжӣҙж–°жңҹй–“з„Ўжі•дҪҝз”ЁCloud ManagerгҖӮ

    жӯҘй©ҹ

    1. еңЁCloud Managerдё»жҺ§еҸ°еҸідёҠи§’гҖҒжҢүдёҖдёӢгҖҢиЁӯе®ҡгҖҚең–зӨәгҖҒз„¶еҫҢйҒёеҸ–гҖҢи»ҹй«”жӣҙж–°гҖҚгҖӮ

    2. жҢүдёҖдёӢйҖЈзөҗгҖҒд»ҘйҡЁйҷ„зҡ„дҝ®иЈңзЁӢејҸжӣҙж–°Cloud ManagerгҖӮ

    3. е®ҢжҲҗзўәиӘҚе°Қи©ұж–№еЎҠгҖҒз„¶еҫҢжҢүдёҖдёӢ*зўәе®ҡ*гҖӮ

    4. йҒёеҸ–жӮЁжүҖжҸҗдҫӣзҡ„дҝ®иЈңзЁӢејҸгҖӮ

    Cloud ManagerжңғеҘ—з”Ёдҝ®иЈңзЁӢејҸгҖӮжӮЁеҸҜд»ҘеңЁе№ҫеҲҶйҗҳеҫҢзҷ»е…Ҙдё»жҺ§еҸ°гҖӮ
